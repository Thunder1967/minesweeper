## 介紹
地雷會隨機分佈在矩形分割而成的方格之中，點擊所有安全的方格即可贏得遊戲。方格內的數字代表鄰近方格有多少地雷，您可根據不同數字推導出特定方格是否含有地雷。點擊滑鼠右鍵放置旗幟以標明地雷位置，點擊左鍵開啟方格。

### 操作
滑鼠左鍵挖開地塊
滑鼠右鍵標定地塊(插旗子)
對插旗處或數字處雙擊左鍵可自動點開地塊周圍未插旗的地塊(可能會引爆地雷)

### 設定
設定中可以調整地圖大小和地雷數量
可開啟提示在遊戲開始時標示一個絕對沒有地雷且臨近8格內都沒有地雷的地塊(黃色地塊)
